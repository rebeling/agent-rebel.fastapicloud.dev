## Security-first AI gateways

- **NeuralTrust TrustGate**: positions itself around centralized AI and agent security, policy, and observability.
- **Netskope AI Gateway**: focuses on intercepting and securing traffic between agents and models.

## Agent and MCP gateways

- **agentgateway**: focuses on connectivity for agents and MCP-style systems, with standalone deployment options.
- **Kong, Zuplo, and TrueFoundry** also increasingly combine LLM and MCP governance.

## Broader AI API platforms

- **WSO2** is adding LLM provider and proxy capabilities to its API platform.
- **New API** is an open-source model aggregation and distribution project.
- **Zenlayer AI Gateway** offers managed unified access to model providers.

## Use this page carefully

These products may be relevant, but this snapshot does not provide a like-for-like feature and deployment audit. Add a dedicated provider page only after checking official architecture, data handling, licensing, and operational documentation.
